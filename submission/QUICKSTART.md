## How to Run

### Option 1: Interactive Jupyter Notebook (RECOMMENDED for exploration)

```bash
cd submission
jupyter notebook exploration_notebook.ipynb
```

This opens an interactive notebook where you can:
- Execute code cells step-by-step
- See visualizations inline
- Read detailed markdown explanations
- Modify and re-run analyses

### Option 2: From Command Line (For quick analysis)

```bash
cd submission
python analysis.py
```

### Option 2: Using Batch File (Windows)

```batch
cd submission
python analysis.py
```

### Option 3: Using Python Directly

```bash
python -m submission.analysis
```

## Expected Output

When you run `python analysis.py`, the script will:

1. **Load the dataset** from `../dataset/train/train.csv`

## Troubleshooting

**Error: "ModuleNotFoundError: No module named 'pandas'"**
→ Run: `pip install pandas numpy matplotlib seaborn scikit-learn`

**Error: "FileNotFoundError: ../dataset/train/train.csv"**
→ Ensure script is run from within the submission/ folder
→ Verify dataset exists at: ../dataset/train/train.csv

**Error: "Permission denied" on PNG files**
→ Ensure write permissions in submission/ directory
→ Try: `chmod -R 755 submission/`

## Dataset Information

The analysis uses the machine predictive maintenance dataset with:
- **10,000 observations** from manufacturing equipment
- **11 columns** including operational and failure metrics
- **Multiple failure types** (TWF, HDF, PWF, OSF, RNF)
- **Balanced data** with both healthy and failed states

## Key Findings Summary

1. **Temperature Impact:** 5% failure rate at low temps → 15% at high temps
2. **Speed Relationship:** Optimal range in mid-speeds; risks at extremes  
3. **Tool Wear:** Strongest predictor; failure risk rises sharply above 120 min
4. **Failure Types:** Tool wear dominant (~40%), heat dissipation secondary (~30%)
